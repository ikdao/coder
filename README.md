# Ikdao Coder

Ikdao Coder, a Indic, Indie online HTML, JS, CSS code editor. that just work!  

[Ikdao Coder](https://ikdao.org/tool/coder/ "Ikdao COder")  

Simple realtime HTML Editor, that let you play with html,js,css online or offline. Available through PWA installation.  

Inspired from [Mr Doob HTML Editor](https://github.com/mrdoob/htmleditor/)  

We implemented Native Compression Stream API intead of rawdeflate/rawinflate libraries of original code, simplified and updated of legacy dependency.
